## 🔧 How It Works

### **Simple Architecture:**

```
Visitor clicks button
    ↓
React component opens popup
    ↓
Visitor types message
    ↓
Sent to /api/chat endpoint
    ↓
Backend tries free AI API
    ↓
Gets response (AI or smart fallback)
    ↓
Response displayed in popup
    ↓
Conversation continues
```

### **Behind the Scenes:**

- **Frontend**: React, TailwindCSS, Framer Motion
- **Backend**: Next.js API route
- **AI**: Free Hugging Face API (with intelligent fallbacks)
- **Database**: None (no data stored)
- **Authentication**: None (completely open)
