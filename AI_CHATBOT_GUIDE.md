## 🔧 How Messages Get Responses

### **Process Flow:**

```
User Types Message
    ↓
Sent to /api/chat endpoint
    ↓
Tries Free Hugging Face API
    ↓
If API available: Returns generated response
    ↓
If API unavailable: Uses smart fallback
    ↓
Response appears in chat window
```

### **Smart Fallback System:**

When the free API isn't available, the chatbot:
1. Analyzes keywords in the message
2. Detects question type (portfolio, tech, greeting, etc.)
3. Responds with relevant pre-written text
4. Feels natural and helpful

## 🆘 Troubleshooting

### ❌ Chatbot button not visible

**Solution:**
- Refresh the page (Ctrl+R)
- Check if JavaScript is enabled
- Try a different browser
- Clear cache (Ctrl+Shift+Delete)

### ❌ Can't type in chatbox

**Solution:**
- Click in the input area
- Make sure cursor is in the box
- Check if any popup blockers active
- Try disabled browser extensions

### ❌ Button doesn't do anything

**Solution:**
- Check browser console for errors (F12)
- Try refreshing page
- Check browser compatibility
- Try incognito mode

### ❌ Response not appearing

**Solution:**
- Wait a moment (may be processing)
- Check internet connection
- Try again with different message
- Check browser console for errors
